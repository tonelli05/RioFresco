# RioFresco
O RioFresco utiliza inteligência de dados e geoprocessamento para combater as ilhas de calor no Rio. Ao cruzar dados de uso do solo, meteorologia e saúde, o projeto mapeia como o déficit de vegetação eleva temperaturas e causa internações respiratórias. O objetivo é otimizar a arborização urbana como política pública de saúde e justiça climática.
